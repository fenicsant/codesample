# Назначение

Приложение представляет собой небольшой черновик для решения судоку.

# Описание применения

Перемещение между ячейками происходит при помощи клавиш управления курсором 
(стрелок) или при помощи левой кнопки мыши.

Каждая ячейка содержит допустимые варианты значений этой ячейки.

Переключение вариантов значений происходит с помощью цифровых клавиш или при 
помощи правой клавиши мыши.

Для определения значения ячейки (единственного допустимого значения), возможны 
два пути: снять все другие допустимые значения; указать нужное число при помощи 
цифровой клавиши с зажатой клавишей «Ctrl» или двойной клик мыши по ячейке.

# Используемые средства

Основным назначением проекта является демонстрация моей способности к разработке 
на Qt.

Проект разработан на с++, используя Qt4, библиотеки QtCore и QtGui. Может 
функционировать на стационарных компьютерах. Минимальное разрешение экрана 
зависит он настроек отображения шрифтов (в моем случае 460х480).

# Описание программы

Программа состоит из трех модулей:

- модуль точки входа в программу (mainproc.cpp);
- модуль окна (mwind.h, mwind.cpp);
- модуль подсказок решения (cellvalidator.h, cellvalidator.cpp).

В связи с небольшим объемом решаемой задачи, и для уменьшения исходного кода, 
в программе применялись только базовые средства наследования классов. Таким 
образом механизмы сигнал-слотов, модели данных и прочее увеличили бы объем 
программы и понизили ее эффективность, и следовательно и увеличило бы время 
разработки.
